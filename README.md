## ⚙️ Getting Started

### Prerequisites

- Node.js
- npm

---

## 📥 Installation

Clone the repository

```bash
git clone https://github.com/Amardeep-Chouhan/Portfolio.git
```

Navigate to the project folder

```bash
cd Portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm start
```


## 🛠 Build for Production

```bash
npm run build
```
